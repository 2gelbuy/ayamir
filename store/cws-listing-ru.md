## Detailed Description
AyaMir — бесплатное расширение Chrome, которое объединяет Помодоро-таймер, менеджер задач, блокировщик сайтов и систему геймификации в одном инструменте. Без аккаунтов. Без облака. Без слежки. Всё на вашем устройстве.

В отличие от StayFocusd, Cold Turkey или BlockSite — AyaMir даёт вам задачи + блокировку + таймер + XP систему в одном расширении. Без подписок, без сбора данных.

КЛЮЧЕВЫЕ ФУНКЦИИ

УМНОЕ УПРАВЛЕНИЕ ЗАДАЧАМИ
- Очередь задач с приоритетами и умной сортировкой
- Редактирование, перенос, напоминания, дедлайны
- Быстрая задача с ЛЮБОЙ страницы — Ctrl+Shift+K (командная палитра)
- Правый клик по странице, ссылке или тексту — сохранить как задачу
- Фокус дня — определи главный приоритет утром

ПОМОДОРО-ТАЙМЕР И РЕЖИМ DEEP WORK
- Таймер фокусировки: 15 / 25 / 45 / 60 / 90 минут
- Круговой прогресс с обратным отсчётом в реальном времени
- Жёсткий режим — нельзя отменить после старта
- Настраиваемые перерывы
- Статистика: общие часы фокуса, завершённые сессии

БЛОКИРОВЩИК САЙТОВ
- Блокировка отвлекающих сайтов во время сессий фокуса
- Быстрые категории: Соцсети (9 сайтов), Новости (7), Развлечения (7), Шоппинг (6)
- Любые домены в чёрный список
- Блокировка по расписанию — авто-блокировка в рабочие часы
- Челлендж ввода текста для обхода — заставляет задуматься

ГЕЙМИФИКАЦИЯ
- Зарабатывайте XP за задачи и сессии фокуса
- 10 уровней: от Новичка до Высшего
- 8 достижений: Ранняя Пташка, Ночная Сова, Демон Скорости
- Ежедневные серии — не ломай цепочку
- Бонусы за выполнение в срок, ранним утром или ночью

100% ПРИВАТНОСТЬ
- Все данные в IndexedDB и Chrome Storage — локально
- Ноль сетевых запросов — работает полностью офлайн
- Без аккаунтов, серверов, аналитики, телеметрии
- Экспорт/импорт данных (JSON бэкап)
- Открытый исходный код — MIT лицензия

ДЛЯ КОГО AYAMIR?
- Студенты — фокус во время учёбы
- Удалёнщики — борьба с отвлечениями в браузере
- Разработчики — менеджер задач с горячими клавишами
- Люди с СДВГ — структурированные сессии фокуса с геймификацией
- Все, кому нужна бесплатная альтернатива StayFocusd, Cold Turkey, Forest или Todoist

ЧЕМ AYAMIR ОТЛИЧАЕТСЯ?
- Бесплатно — все основные функции без рекламы
- Безлимитная блокировка — блокируй сколько хочешь сайтов (без ограничения в 3 сайта как у BlockSite)
- Без аккаунта — работает за 10 секунд после установки
- Всё-в-одном — задачи + Помодоро + блокировщик + геймификация
- 100% офлайн — данные не покидают браузер
- Открытый код — MIT лицензия, проверяй сам

Без рекламы. Без слежки. Только фокус.
