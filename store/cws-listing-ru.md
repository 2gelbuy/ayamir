## Detailed Description
AyaMir — Береги время, строй наследие.

Бесплатное расширение для продуктивности, которое объединяет менеджер задач, Помодоро-таймер, блокировщик сайтов и систему геймификации в одном инструменте. Без аккаунтов. Без облака. Без слежки. Всё на вашем устройстве.

В отличие от StayFocusd, Cold Turkey или BlockSite — AyaMir даёт вам задачи + блокировку + таймер + XP систему в одном расширении. Без подписок, без сбора данных.
