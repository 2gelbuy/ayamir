## 2. Unique Selling Points of Top Competitors

### BlockSite (5M+ users)
- Visual block pages with motivational content
- Group blocking (block by category)
- Work mode scheduling
- Website redirects (blocked site → chosen site)
- **Weakness**: Free tier hard-limited to 3 sites — #1 user complaint

### StayFocusd
- Nuclear Option (locks settings for set time — no override)
- Time-budget approach (X minutes per day on blocked sites)
- Challenge to disable (requires typing long paragraph)
- **Weakness**: No task management, no Pomodoro, outdated UI

### Forest
- Visual tree grows during focus sessions (gamification)
- Real tree planting partnership (Trees for the Future)
- Social aspect: grow forest with friends
- **Weakness**: 3.8★ despite 900K users — significant user frustration
- Top complaints: premium paywall for basic features, sync issues, no site blocking in extension

### Focus To-Do
- Native Pomodoro + task list integration — best-in-class execution
- Cloud sync across devices
- Statistics and productivity reports
- **Weakness**: Requires account creation, cloud dependency, paid tier for full sync

### Freedom
- Multi-device sync (Chrome + Firefox + macOS + iOS + Android)
- Session scheduling and recurring schedules
- Locked mode
- **Weakness**: $8.99/mo subscription, no Pomodoro, no tasks

### LeechBlock NG
- Highly configurable time rules per site
- Open-source, privacy-conscious
- **Weakness**: Power-user complexity, no Pomodoro, no tasks, no gamification

## 4. Pricing Models

| Model | Extensions Using | Conversion Rate |
|-------|-----------------|-----------------|
| **Completely free** | StayFocusd, Marinara, LeechBlock NG | N/A (donation/nothing) |
| **Freemium (feature gate)** | BlockSite (3-site limit), Forest (cloud sync gated), Focus To-Do (stats gated) | Industry avg 2.8%, top 5-8% |
| **Subscription** | Freedom ($8.99/mo), Cold Turkey ($39/yr) | Converts well with cross-device value prop |
| **One-time** | Cold Turkey Pro ($39 lifetime) | Strong conversion vs subscription for extensions |
| **Open-source / MIT** | LeechBlock NG, (AyaMir) | Community trust, but no revenue |

### Freemium gate strategies observed:
- **Site count limit** (BlockSite: 3 sites free) — creates clear pain at scale
- **Sync/cloud** (Focus To-Do, Forest) — local use free, sync paid
- **Stats/history** (Focus To-Do) — basic free, analytics paid
- **Schedule** (BlockSite, Freedom) — manual blocking free, auto-schedule paid

### AyaMir monetization path (recommended at 5K users):
- Gate: Cloud backup/sync or team features (keep local usage 100% free)
- Do NOT gate site count, Pomodoro, or gamification — these are differentiators
- Consider one-time purchase ($9-15) over subscription — lower friction for privacy users

## 5. CWS Listing Format Analysis

### Name/Title Format (128 char max)
Best practice: `[Brand] - [Primary Keyword], [Secondary Keyword] & [Tertiary Keyword]`
- AyaMir current: "AyaMir - Pomodoro Timer, Task Manager & Site Blocker" — correct format, strong keywords

### Short Description (132 chars max)
Top performers lead with the #1 use case, include 2-3 keywords, end with differentiator:
- AyaMir current: "Pomodoro timer, task manager & site blocker in one extension. 100% local — no account, no cloud, no tracking. Earn XP, stay focused." — well-structured

### Detailed Description Format Patterns
1. **Hook paragraph** — 2-3 sentences, all keywords, main differentiator
2. **Competitor comparison line** — "Unlike X, Y, Z — AyaMir gives you..."
3. **Feature sections in CAPS** — CAPS headers get eye-scanned, increase dwell time
4. **Bullet points** — 3-6 bullets per section, action verbs
5. **Privacy/trust section** — increasingly important, especially post-GDPR
6. **WHO IS IT FOR section** — helps CWS categorize and recommend
7. **Call to action / closing line** — short, punchy

### Screenshot Best Practices (1280x800 or 640x400, PNG)
- First screenshot must show core functionality (most important — drives 60%+ of impression-to-install)
- Text overlays on screenshots: +23% install rate vs bare screenshots
- Recommended overlay pattern: headline (benefit) + 2-3 bullet points
- Dark theme or clean white — avoid cluttered UI screenshots
- Show the popup open (most extensions show their main UI)
- 5 screenshots max — use all 5

### Promo Tile (440x280 small, 1400x560 marquee)
- Marquee tile required for Featured badge consideration
- Pattern: brand logo left, product screenshot right, 1 headline, solid bg
- AyaMir has marquee tile created — ensure it's uploaded at publish

### Category Selection Impact
- "Productivity > Workflow & Planning" — correct choice, lower competition than generic "Productivity"
- AyaMir is correctly positioned here

- Tagline formula: "[What it does] — [Key differentiator]"
- Maker comment posted immediately at launch explaining the "why"
- Ask existing users (friends, family, GitHub stars) to upvote on launch day
- Tag: Chrome Extension, Productivity, Pomodoro, Open Source

### Medium-term (month 2-3)
9. **Publish SEO blog post** targeting "best pomodoro chrome extension 2026" — highest volume keyword in the space
10. **Create "BlockSite Alternative" landing page section** — 5M+ BlockSite users frustrated by 3-site limit are an ideal audience

### Positioning to double down on

- Do NOT gate site count (will create the same backlash as BlockSite)
