## Detailed Description
AyaMir — Protect your time, build your legacy.

The free productivity Chrome extension that combines smart task management, Pomodoro timer, website blocker, and gamification into one powerful focus toolkit. No accounts. No cloud. No tracking. Everything stays on your device.

Unlike StayFocusd, Cold Turkey, or BlockSite — AyaMir gives you task management + site blocking + focus timer + XP system all in one extension. No subscriptions, no data collection, no limits.

━━━━━━━━━━━━━━━━━━━━

SMART TASK MANAGEMENT
• Priority-based task queue with smart sorting algorithm
• Inline editing, snooze reminders, and scheduled tasks
• Quick add task from ANY page — press Ctrl+Shift+K
• Right-click any page, link, or selected text to save as task
• Daily focus goal — set your #1 priority each morning

POMODORO TIMER & DEEP WORK MODE
• Focus timer with presets: 15 / 25 / 45 / 60 / 90 minutes
• Circular SVG progress ring with real-time countdown
• Hard Lock Mode — cannot cancel once started (for serious focus)
• Break timer with customizable duration
• Session tracking: total hours, completed sessions
• Works like a Pomodoro timer but built into your browser

WEBSITE BLOCKER & DISTRACTION BLOCKER
• Block distracting websites during focus sessions
• One-click category presets: Social Media (9 sites), News (7), Entertainment (7), Shopping (6)
• Add any custom domain to your blocklist
• Scheduled blocking — auto-block during work hours (configurable days/times)
• Typing challenge to bypass — forces you to type a motivational quote to continue
• Focus mode blocks sites even outside Deep Work sessions

GAMIFICATION & MOTIVATION
• Earn XP for completing tasks and focus sessions
• 10 levels: Beginner → Novice → Apprentice → Expert → Master → Grandmaster → Legend → Mythic → Transcendent
• 8 achievements: Early Bird, Night Owl, Speed Demon, Perfect Timing, and more
• Daily streaks — don't break the chain
• Points bonuses for completing tasks on time, early morning, or late night

100% PRIVATE — ZERO DATA COLLECTION
• All data stored locally in IndexedDB and Chrome Storage
• Zero network requests — works completely offline
• No accounts, no servers, no analytics, no telemetry
• Full data export/import (JSON backup)
• Open source — MIT License — audit the code yourself on GitHub

KEYBOARD SHORTCUTS
• Ctrl+Shift+E — Open AyaMir popup
• Ctrl+Shift+K — Quick task from any page (command palette)
• N or / — New task • D — Deep Work • V — Stats • S — Settings • ? — Help

LANGUAGES
Available in: English, Russian, Spanish, Chinese (Simplified), Hindi
